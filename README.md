# Secure Multi-Vendor E-Commerce Marketplace Platform

## 🔹 Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: PostgreSQL

## 🔹 Features
- User Authentication (Login/Register)
- Product Management
- Cart System
- Order Management
- Multi-Vendor Support

## 🔹 Project Structure
backend/
frontend/

## 🔹 How to Run

### Backend
- Open in IntelliJ
- Run Spring Boot Application

### Frontend
cd frontend
npm install
npm start

## 🔹 Demo
